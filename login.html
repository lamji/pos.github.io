1.Login Screen

Kasama sa login screen ang mga sumusunod:

  • • Input ng Email: Ilagay ang iyong rehistradong email.
  • • Input ng Password: Ilagay ang iyong password.
  • • Login Button: I-tap upang mag-login.
  • • Nakalimutang Password: Gamitin ito upang i-reset ang iyong password.
  • • I-activate ang Account: I-tap ito kung wala ka pang account.
2. Pag-activate ng Account

Kung wala ka pang account, sundin ang mga hakbang na ito:

  • 1. I-tap ang I-activate ang Account sa Login screen.
  • 2. Magbigay ng mga sumusunod na impormasyon:
    • - Email Address: Ilagay ang iyong wastong email.
    • - Activation Code: Ilagay ang gusto mong code.
  • 3. Kapag matagumpay, magkakaroon ka ng 7-days na trial.
  • 4. Bumalik sa Login screen upang mag-login.
3. Pag-login

Kapag na-activate na ang iyong account:

  • 1. Ilagay ang iyong email sa field ng Email.
  • 2. Ilagay ang iyong code sa field ng Password.
  • 3. I-tap ang Login button.
4. Nakalimutang Password?

Kung nakalimutan mo ang iyong password, gamitin ang opsyong Nakalimutang Password:

  • 1. I-tap ang forgot Password button.
  • 2. Sundin ang mga tagubilin upang i-reset ang iyong password.
  • 3. Gamitin ang iyong bagong password upang mag-login.
